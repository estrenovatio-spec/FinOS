# Тексты помощи (русский)

**Как сохранить в приложение:** правьте этот файл, затем напишите в чате **«примени тексты помощи»** — перенесём в код и задеплоим.

Или сразу правьте в коде:
- шпаргалка и база ИИ → `src/lib/help-faq-content.ts`
- подписи экрана → `src/lib/i18n.ts` (ключи `help*`)
- тон ответов ИИ → `src/lib/help-chat.ts` → `HELP_CHAT_SYSTEM`

*Последняя синхронизация из этого файла в код — применена.*

## 1. Экран в приложении (Настройки → Помощь)

**Шпаргалка (список)**
```
/start, /help — справка в боте
Текст в приложении или голос/текст боту — «потратил 500 на обед»
Копилка — «отложил 5000 на отпуск»
Mini App — бот → «Открыть приложение»
Партнёр вдвоём — Облако → Вдвоём → код → Присоединиться
Подробнее — спросите ИИ ниже
```

**Заголовок шпаргалки**
```
Шпаргалка
```

**Кнопка в настройках**
```
Помощь и вопросы
```

**Заголовок окна**
```
Помощь — Бюджет
```

**Заголовок блока ИИ**
```
Спросите ИИ
```

**Подсказка под заголовком**
```
Здесь можно спросить только ИИ: как пользоваться приложением и по вашим тратам. С облаком — ответ по базе на сервере.
```

**Плейсхолдер в поле ввода**
```
Например: как пригласить партнёра? Сколько потратил на еду за месяц?
```

**Кнопка отправки (aria)**
```
Спросить
```

**Пока ИИ думает**
```
Ищу ответ…
```

**Пустой чат (до первого вопроса)**
```
Напишите вопрос — про программу (команды, облако, голос) или про ваши финансы. Лимита нет.
```

**Подпись: ответ из облака**
```
Ответ по данным из облачной базы.
```

**Подпись: ответ с телефона**
```
Ответ по данным на этом устройстве (облако не подключено).
```

**Если ИИ недоступен (сервер)**
```
Сейчас ИИ недоступен. Попробуйте задать вопрос позже.
```

**Ошибка сети**
```
Ошибка сети. Проверьте интернет.
```

**Не удалось получить ответ**
```
Не удалось получить ответ. Попробуйте переформулировать.
```

## 2. Сообщение бота /start и /help

```
Бюджет — справка бота

Команды
/start · /help — это сообщение
Любой текст без / = запись операции

Примеры
• потратил 500 на обед
• зарплата 80000
• отложил 5000 на отпуск

Голос — голосовое в чат; или тот же текст.
Синхронизация с Mini App при включённом облаке.

Справка по приложению — Mini App → Настройки → Помощь → спросите ИИ (без лимита)

Бот: @Fin_BU_bot
```

*(В Telegram часть строк выделяется жирным через HTML — в коде `formatBotHelpHtml`.)*

## 3. База знаний для ИИ (на это опираются ответы в чате)

### С чего начать
- Откройте бота @Fin_BU_bot → «Открыть Mini App».
- Запишите трату: текст + «Добавить» в приложении или голосовое/текст боту.
- Вход только через Telegram — пароль не нужен.

### Команды бота
- /start и /help — краткая справка и кнопка приложения.
- Других команд нет. Любой текст без "/" — запись операции (как голосовое).
- Примеры: потратил 500 на обед · зарплата 80000 · отложил 5000 на отпуск

### Как записывать
- В приложении: текст + «Добавить».
- В боте: голосовое или текст — синхронизация с приложением при включённом облаке.
- Список операций: редактирование и удаление на главном экране.

### Примеры фраз
- Расход: 500 кофе · потратил 1500 в пятёрочке
- Доход: зарплата 80000 · получил 5000 фриланс
- Копилка: отложил 5000 на отпуск · закинул 2000 в подушку
- Доход + копилка: зарплата 100000, 20000 на отпуск
- Партнёр: Маша 800 продукты (имя как в настройках)
- Поддерживаются «1,5 млн», «сто тысяч». В фразе нужна цифра.
- Категория — по словам (пятёрочка → Продукты). Свои слова — Настройки → Категории.

### Партнёр и семья
- Без облака: укажите имя партнёра в настройках — метки «Я» / партнёр.
- Облако «Веду один» — бэкап на сервере, один Telegram.
- «Вдвоём» — код из 6 символов, второй: Настройки → Присоединиться.
- Фильтры Общий / Я / Партнёр — на главном экране.

### Облако и синхронизация
- Без облака данные только на этом устройстве.
- С облаком новые операции уходят автоматически.
- Синхронизировать · Скачать с облака · Отправить на облако — в блоке «Облако и семья».
- Браузер: Войти через Telegram → Подключить этот браузер → Скачать с облака.
- Партнёр не видит траты — не присоединился по коду или открывает не из Telegram.

### Подписка
- При включённой оплате: облако + голосовой бот требуют активную подписку.
- Локальные данные на телефоне не удаляются при паузе.
- Оплата в настройках (Облако) → «Я оплатил — обновить».
- Точная цена и срок — в приложении (500 ₽ / 30 дн.).

### Цели и планирование
- На главной: Копилки · Лимиты · Подушка · Регулярные платежи.
- День начала месяца (зарплата) — в блоке планирования.
- Голосом: отложил 5000 на отпуск · создать цель отпуск цель 150000

### Советы и AI
- Мини-советы, разбор недели и месяца, чат по отчёту — блок «Советы и AI».
- AI — общие подсказки по бюджету, не налоги и не инвестиции.
- Если AI недоступен, операции всё равно записываются.

### Не работает?
- Не нашёл сумму — добавьте цифру: 500 обед.
- Пусто в браузере — войти через Telegram, скачать с облака.
- Зависло — «Сбросить кэш и перезагрузить» в настройках (если есть).
- Оплата не применилась — «Я оплатил — обновить», перезапустить Mini App.

### Данные и очистка
- Локально — данные не на сервере. В облаке — привязка к Telegram ID.
- «Очистить данные» — только это устройство; в облаке остаётся.
- После очистки можно снова скачать с облака.

## 4. Инструкция для ИИ (тон ответов, не показывается пользователю)

```
Ты помощник приложения «Бюджет» (VoiceBudget) — учёт доходов и расходов в Telegram.

Отвечай по-русски. Кратко (2–4 абзаца или список). Дружелюбно, без осуждения.

У пользователя НЕТ других экранов справки — все вопросы «как пользоваться» только из блока APP HELP.
Про деньги — только из JSON с его операциями.

Правила:
- Приложение, бот, облако, подписка, голос, категории, планирование → только APP HELP.
- Траты, доходы, тренды, цели → только USER FINANCES; цифры не выдумывать.
- Не давать налоговых, юридических и инвестиционных советов.
- Если спрашивают про траты, а записей нет — коротко объяснить и предложить записать несколько операций.
```
