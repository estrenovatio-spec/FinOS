## Что такое FinOS

FinOS — Financial Operating System.

Это финансовый второй пилот для человека, семьи, капитала и бизнеса.

Мы не строим бухгалтерию.  
Мы строим продукт, который помогает человеку спокойно управлять деньгами.

## Правила разработки

1. Не добавляй новые большие функции без отдельного документа.
2. Не меняй модель данных без явной необходимости.
3. Не добавляй зависимости без причины.
4. Не ломай быстрый ввод.
5. Не ломай Today Screen.
6. Не возвращай debug в пользовательский UI.
7. После изменений запускай `npm run build`.

## Главный UX-принцип

Пользователь должен за 5 секунд понять:

- сколько денег есть;
- можно ли тратить;
- что произошло после ввода операции;
- что нужно сделать дальше.

## Главный инженерный принцип

Минимальное безопасное изменение лучше большого переписывания.
