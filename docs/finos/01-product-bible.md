# Product Bible

## Позиционирование

FinOS — операционная система денег.

Внешний бренд может называться «Просто Бюджет».  
Внутренняя архитектура продукта — FinOS.

## Для кого

1. Новичок — хочет понять, куда уходят деньги.
2. Семья — хочет видеть общий бюджет, цели и платежи.
3. Продвинутый пользователь — хочет управлять долгами, подушкой, фондами, инвестициями.
4. Предприниматель — хочет видеть бизнес, прибыль и вывод денег домой.

## Главная ценность

Не учёт ради учёта, а финансовое спокойствие.

## Главная метрика

Financial Peace Index — насколько спокойно человек чувствует себя относительно денег.
